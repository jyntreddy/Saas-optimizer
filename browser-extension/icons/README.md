To create icons:
1. Design a 128x128 icon with your logo
2. Resize to 48x48 and 16x16
3. Save as icon128.png, icon48.png, icon16.png in this folder

Or use online tools like:
- https://favicon.io/
- https://realfavicongenerator.net/

Recommended design:
- Simple "$" or "💰" symbol
- Blue/green gradient background
- Clear and recognizable at small sizes
