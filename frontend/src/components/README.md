# Components Directory

This directory contains all React components organized by feature:

## Structure

- `analytics/` - Analytics and chart components
- `dashboard/` - Dashboard-specific components
- `layout/` - Layout components (headers, sidebars, etc.)
- `subscriptions/` - Subscription-related components
- `ui/` - Reusable UI components (buttons, inputs, modals, etc.)
- `common/` - Common/shared components

## Component Guidelines

1. Use functional components with TypeScript
2. Keep components small and focused
3. Use composition over inheritance
4. Extract reusable logic into custom hooks
5. Follow naming conventions (PascalCase for components)
6. Include PropTypes or TypeScript interfaces
7. Add comments for complex logic
